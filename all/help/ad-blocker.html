Advertisement/Malware Blocker

This applet uses DNS resolution to block known addresses, allowing Guardian Devil to act as an advertisement and malware blocker. Anytime that your connected devices request the IP address of a known malicious website or advertisement source, Guardian Devil denies these requests, causing those advertisements to not be loaded. Here, you can change the blocklists as well.


This feature may slightly improve your connection speed. You can make changes or disable this feature using the Advertisement Blocker page.
