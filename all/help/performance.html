Performance

This applet allows you to see visualizations of your internet speed. Guardian Devil will regularly run speed tests and add the output to the graphs, displaying download speed, upload speed, and latency. You can change the frequency of these tests, or disable them entirely, within the applet.


You may also run a manual speed test, useful for diagnosing network problems.
